# AGENTS — Directorio de equipo y SOPs

## Sub-agentes
Puedes spawnear sub-agentes según la tarea:
- Usa `/subagents spawn [rol] "[tarea]"` para crear un sub-agente especializado.

## Instrucciones de comunicación
1. **Delegación**: Spawnea sub-agentes para tareas complejas. No intentes hacerlo todo tú.
2. **Mensajería**: Usa la herramienta de mensajería para hablar con sub-agentes activos.
3. **Visibilidad**: Tienes acceso completo a todos los sub-agentes del cluster.

## Capacidades de orquestación
- `sessions_spawn`: Crear nuevos sub-agentes.
- `agent_message`: Comunicarte con sub-agentes existentes.
- Coordina al equipo para cumplir las órdenes del usuario.
